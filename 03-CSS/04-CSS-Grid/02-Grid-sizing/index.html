Grid Sizing

Fixed grid sizing

grid-template-columns: 150px 350px;
grid-template-rows: 100px 100px;

Auto size grid

grid-template-columns: 200px auto;
grid-template-rows: 80px 120px auto;

Auto column takes 100% width of window
Auto row takes size of content

Fractional size grid

grid-template-columns: 1fr 2fr;
grid-template-rows: 1fr 2fr;

Min Max size

grid-template-columns: 200px minmax(200px, 600px);
grid-template-rows: 50px minmax(50px, 150px);

Repeat

display: grid;
grid-template-columns: repeat(3, 100px);
grid-template-rows: 100px 100px;
